# Sistema de Cadastro de Usuários

### Este é um sistema simples de cadastro de usuários desenvolvido utilizando Laravel, Bootstrap e MySQL.

## Pré-requisitos

- **PHP** (>= 8.0)
- **Composer**
- **MySQL**
- **Node.js** e **NPM**
- **Laravel** (>= 10)
